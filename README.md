# 拼豆辅助工具

一个用于拼豆制作的辅助网站，包含颜色识别和色卡查看两个主要工具。

## 功能特性

### 1. 颜色识别工具
- **图片上传**：支持上传图片文件
- **颜色选择**：点击图片上的任意位置即可识别该位置的颜色
- **图片缩放**：支持放大和缩小图片，方便查看细节
- **颜色匹配**：自动从Mard221色表中找出最接近的颜色
- **相似度显示**：显示匹配颜色的相似度百分比
- **多选项**：提供最接近的1个颜色和另外10个相对接近的颜色供选择

### 2. 色卡工具
- **分类查看**：按A-H和M组分类显示所有颜色
- **详细信息**：显示每个颜色的ID、HEX值和RGB值
- **过滤功能**：可以按组别筛选查看
- **美观界面**：简洁大方的卡片式布局

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 使用顶部导航切换不同的工具
3. 在颜色识别工具中：
   - 点击"上传图片"选择图片
   - 点击图片上的位置识别颜色
   - 使用放大/缩小按钮调整图片大小
   - 点击"重置"重新上传图片
4. 在色卡工具中：
   - 使用顶部按钮切换不同的颜色组
   - 查看每个颜色的详细信息

## 文件结构

- `index.html` - 主页面
- `styles.css` - 样式文件
- `script.js` - 主要功能逻辑
- `colors.js` - 颜色数据定义和匹配算法

## 技术说明

- 使用纯JavaScript实现，无需额外依赖
- 颜色匹配算法基于RGB空间的欧氏距离计算
- 响应式设计，支持不同屏幕尺寸
- 现代化的UI设计，使用渐变和阴影效果
