## 📖 项目简介

拼豆辅助工具是一个基于 Web 的智能颜色识别系统，专为拼豆（Perler Beads）制作爱好者打造。通过先进的颜色匹配算法，帮助您快速识别图片中的颜色，并找到最匹配的拼豆色号，让您的创作过程更加高效便捷。

### ✨ 核心优势

- 🎯 **精准识别** - 基于 RGB 空间欧氏距离的智能颜色匹配算法
- 🎨 **多色号系统** - 支持 MARD、COCO、漫漫、盼盼、咪小窝 等多种色号系统
- 📱 **响应式设计** - 完美适配桌面端和移动端设备
- 🌓 **主题切换** - 支持浅色/深色主题，保护您的眼睛
- 💾 **颜色管理** - 自定义颜色库，导入导出配置，轻松管理您的拼豆库存
- 🚀 **零依赖** - 纯 JavaScript 实现，无需安装，打开即用

## 🚀 快速开始

### 方法一：直接使用（推荐）

1. 下载或克隆本项目到本地
2. 用浏览器打开 `index.html` 文件
3. 开始使用！

### 方法二：本地服务器

如果您想通过 HTTP 协议访问（推荐用于开发）：

```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 或使用 Node.js (需要先安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 📚 使用指南

### 🎨 颜色识别工具使用步骤

#### 第一步：上传图片

您可以通过以下三种方式上传图片：

1. **点击上传**
   - 点击"📷 上传图片"按钮
   - 在文件选择器中选择图片文件

2. **拖拽上传**
   - 将图片文件从文件管理器拖拽到上传区域
   - 松开鼠标即可完成上传

3. **粘贴上传**
   - 复制图片到剪贴板（截图、复制图片等）
   - 在上传区域按 `Ctrl+V` (Windows) 或 `Cmd+V` (Mac)

#### 第二步：调整图片

- **缩放图片**
  - 使用鼠标滚轮向上/向下滚动
  - 点击"🔍+ 放大"或"🔍- 缩小"按钮
  - 在移动设备上使用双指捏合手势

- **移动图片**
  - 缩放后，点击并拖拽图片查看不同区域

#### 第三步：识别颜色

1. 在图片上点击您想要识别的颜色位置
2. 右侧将显示：
   - **选中的颜色** - 显示该位置的 HEX 和 RGB 值
   - **匹配的拼豆颜色** - 显示最接近的 1 个颜色和 10 个相似颜色
   - 每个匹配颜色显示：
     - 颜色预览方块
     - 色号（根据当前选择的色号系统）
     - 颜色名称
     - 相似度百分比

#### 第四步：切换色号系统

1. 在页面顶部找到"色号系统"下拉菜单
2. 选择您需要的色号系统（MARD、COCO、漫漫、盼盼、咪小窝）
3. 匹配结果会自动更新为对应系统的色号

### 📋 色卡工具使用步骤

1. **切换工具**
   - 点击顶部导航栏的"色卡"按钮

2. **浏览颜色**
   - 在色卡网格中浏览所有颜色
   - 每个颜色卡片显示颜色预览和基本信息

3. **查看详情**
   - 点击任意颜色卡片
   - 在弹出的详情窗口中查看：
     - 完整的颜色信息（HEX、RGB、色号、名称）
     - 替代颜色推荐

### 🎨 颜色管理使用步骤

1. **打开颜色管理**
   - 点击页面顶部的"🎨 管理我的颜色"按钮

2. **选择颜色**
   - 在颜色网格中勾选您拥有的颜色
   - 使用"全选"或"全不选"按钮快速操作

3. **保存配置**
   - 点击"💾 保存"按钮保存您的选择
   - 配置会自动保存到浏览器本地存储

4. **导出配置**
   - 点击"📤 导出配置"按钮
   - 将颜色配置保存为 JSON 文件

5. **导入配置**
   - 点击"📥 导入配置"按钮
   - 选择之前导出的 JSON 文件
   - 配置会自动加载并应用

### 🌓 主题切换

- 点击页面右上角的月亮图标（🌙）切换到深色主题
- 再次点击太阳图标（☀️）切换回浅色主题
- 主题偏好会自动保存

## 🛠️ 技术说明

### 技术栈

- **前端框架**：纯 JavaScript（ES6+）
- **样式**：CSS3（包含渐变、阴影、动画等现代特性）
- **颜色算法**：基于 RGB 空间的欧氏距离计算
- **存储**：LocalStorage（浏览器本地存储）

### 核心算法

#### 颜色匹配算法

使用 RGB 空间的欧氏距离公式计算颜色相似度：

```
距离 = √[(R₁-R₂)² + (G₁-G₂)² + (B₁-B₂)²]
相似度 = (1 - 距离/最大距离) × 100%
```

其中：
- `R₁, G₁, B₁` 是图片中选中颜色的 RGB 值
- `R₂, G₂, B₂` 是拼豆颜色的 RGB 值
- 最大距离为 `√(255² + 255² + 255²) ≈ 441.67`

### 性能优化

- **防抖（Debounce）** - 优化频繁触发的事件（如窗口调整大小）
- **节流（Throttle）** - 限制高频率事件的执行（如滚动、拖拽）
- **RequestAnimationFrame** - 优化动画和渲染性能
- **事件委托** - 减少事件监听器数量
